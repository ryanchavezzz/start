Shadows
Three elevation levels. All include an inset ring to simulate a border from inside, keeping DOM borders free for interaction states. Dark mode uses white-tinted inset + heavier drop.
--shadow-1
Chips, small UI elements
Resting / default state
--shadow-2
Buttons (.btn)
Primary interactive elements
--shadow-3
Hover state, modals
Search overlay, lifted cards

Motion
A single easing curve and duration pair covers all transitions. Hover the square to feel the easing.
--ease
cubic-bezier(0.4, 0, 0.2, 1)
--t
0.15s var(--ease)
--ease: cubic-bezier(0.4, 0, 0.2, 1); --t: 0.15s var(--ease); /* usage */ transition: background var(--t), border-color var(--t), color var(--t);
Components
Buttons .btn
Every link in the startpage is an .btn. Four icon variants exist: silhouette mask (.si), colored background (.si-color), inline SVG (.svg-icon), and favicon (.fav). Hover to see the lift + shadow-3.
.si — monochrome mask (inherits --text color)
GitHub YouTube Vercel NextDNS
.svg-icon — inline SVG
Bitwarden Google Drive
.fav — Google favicon (grayscale → color on hover)
TRaSH Guides Hacker News
.btn { height: 36px; padding: 8px 14px; font-size: 14px; letter-spacing: -0.28px; border-radius: var(--r-md); box-shadow: var(--shadow-2); transition: box-shadow 0.15s var(--ease), transform 0.15s var(--ease); } .btn:hover { box-shadow: var(--shadow-3); transform: translateY(-1px); }

Holiday Badge
A split pill badge shown in the hero when a holiday is ≤7 days away. Left side shows the holiday name (muted), right side shows the countdown (text color, separated by a border). On the day itself, only the name shows (--text, weight 500).

Search Modal
Triggered by pressing /. Builds its index at runtime by scraping all .btn elements. Arrow keys navigate, Enter opens in new tab, Esc closes. The backdrop uses backdrop-filter: blur(12px).
